## Team Information
- **Team Number:** 23007

- **Purpose:** Competition robot design, visualization, and team documentation

## Features
- Complete 3D model of competition robot
- Team-branded color scheme (Orange/Red)
- Animated demonstrations
- High-quality renders for documentation and presentations

## File Details
- **Main Model:** `FinalVersionOfTheRobot.blend` - Primary Blender project file
- **Renders:** Multiple PNG outputs showing different angles and configurations
- **Animation:** MP4 video demonstrating robot functionality
- **Assets:** Team color schemes and branding materials

## Requirements
To view or edit the 3D model:
- **Blender** (version compatible with the .blend file)
- Recommended: Blender 2.8 or newer

## Usage
1. Download and install [Blender](https://www.blender.org/download/)
2. Open `FinalVersionOfTheRobot.blend` in Blender
3. Explore the model, animations, and scene setup
4. Use for team presentations, engineering documentation, or design reviews
